# IMad_Project


> IMAD 프로젝트 iOS 레포지토리입니다.
> 
> IMAD는 영화/드라마/애니메이션 등 여러 미디어 작품에 대한 리뷰를 남길 수 있고 커뮤니티로 소통할 수 있는 어플입니다.

### 관련링크

- [IMAD 서버](https://github.com/NCookies/imad-server)
- [IMAD iOS 위키](https://github.com/QuaRang1225/IMad_Project/wiki)

#### 언어 및 툴

- Swift
- xcode
  
#### 내부 프레임워크

- SwiftUI
- UIkit
- PhotosUI
- Combine

#### 외부 라이브러리

- Alamofire
